Instance Server
===============

Container Management within Cloud Instance in Synrc PaaS. 

![Login](http://voxoz.com/images/voxoz-dashboard.png)

Overview
--------

Node Server manage LXC containers on Cloud Instance using Docker.

* Login
* Create
* Start
* Stop
* Destroy
* Commit
* Push
* Logout
